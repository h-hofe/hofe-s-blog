一、房价预测（回归分析）

1.1 环境

  1. python(3.6)

  2. matplotlib (3.1.3)

  3. numpy (1.18.1)

  4. pandas (1.0.0)

  5. scikit-learn (0.22.2.post1)

数据集点击下载

1.2 功能

  1. 进行必要的数据预处理和特征工程

  2. 应用两种以上回归分析算法建立预测模型（不限编程语言）

  3. 随机将数据划分为训练集和测试集，进行模型训练和测试

  4. 应用交叉验证法训练和测试预测模型

  5. 评价第(3)和(4)所建立预测模型的性能（用MSE（均方误差）和R2 Score（R2决定系数）等指标评价）

二、结果

逻辑回归的准确率一直不高；多项式回归准确率还可以，这里用的只是1阶，2阶就开始过拟合了，可以用交叉验证看误差在多少。数据集共有2000条，图片只显示前100条。
